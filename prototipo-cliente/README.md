# Prototipo Cliente (React + Vite)

Vista web para validar UI/UX del panel Cliente y como referencia visual para implementación en Filament.

## Desarrollo

```bash
npm install
npm run dev
```

## Notas para desarrolladores

- **Storybook** (`/storybook/...`): criterios junto a cada bloque de ejemplo (p. ej. decisión primario / neutro / secundario en `dash-glass-hero`) y en paneles concretos.
- **UX** (`/ux/...`): contexto en `UxHero` (`guidance`), notas por pestaña donde aplica, y al abrir un **slide-over CRUD** verás la justificación del patrón dentro del propio componente.
- Convenciones backend: `.cursor/rules/uxui.mdc` en la raíz del monorepo.
